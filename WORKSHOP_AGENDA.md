# Google ADK Workshop: Building AI Agents for Healthcare
## Non-Clinical Use Cases - No PHI/PII Processing

**Target Audience:** Novice to Intermediate healthcare professionals
**Duration:** Half-day (4 hours) or Full-day (6-7 hours) workshop options
**Focus:** Visual Agent Builder for accessible, code-light development

---

## Half-Day Workshop (4 hours)

### Session 1: Introduction & Setup (45 minutes)

**9:00 - 9:15 AM: Welcome & Introductions**
- Workshop objectives
- Use case overview
- Ground rules (no PHI/PII)
- Icebreaker: What repetitive task would you automate?

**9:15 - 9:35 AM: What are AI Agents?**
- Definition and concepts
- Real-world examples
- When to use agents vs. traditional automation
- Healthcare-specific opportunities (non-clinical)

**9:35 - 9:45 AM: Google ADK Overview**
- What is ADK?
- Why Google built it
- Visual Agent Builder demo
- Key features and capabilities

**Break: 10 minutes**

### Session 2: Building Your First Agent (75 minutes)

**9:55 - 10:05 AM: Visual Builder Walkthrough**
- Interface tour
- Creating an agent
- Testing and debugging
- Export options

**10:05 - 10:50 AM: Hands-on Exercise 1**
- **Exercise:** Build an HR FAQ Agent
- Guided walkthrough with pauses for questions
- Individual work with instructor support
- Test and iterate

**10:50 - 11:00 AM: Share & Discuss**
- Volunteers demo their agents
- What worked well?
- What was challenging?
- Q&A

**Break: 15 minutes**

### Session 3: Adding Tools & Multi-Agent Systems (90 minutes)

**11:15 - 11:30 AM: Introduction to Tools**
- What are tools?
- Built-in vs custom tools
- When to use tools
- Tool safety and permissions

**11:30 - 12:10 PM: Hands-on Exercise 2**
- **Exercise:** Training Assistant with Tools
- Build custom tools
- Integrate with agent
- Test tool usage
- Debug common issues

**12:10 - 12:25 PM: Multi-Agent Concepts**
- Why use multiple agents?
- Orchestration patterns (sequential, parallel, hierarchical)
- Visual Builder for multi-agent
- Real-world examples

**12:25 - 12:55 PM: Quick Multi-Agent Demo**
- **Instructor-led:** Newsletter creation system
- Watch agents collaborate
- See Visual Builder connections
- Discussion of workflow design

**12:55 - 1:00 PM: Wrap-up & Next Steps**
- Key takeaways
- Resources for continued learning
- How to access workshop materials
- Q&A

**4:20 - 4:40 PM: Real-World Integration**

- API integration

**4:40 - 5:00 PM: Your Use Case Workshop**
- Brainstorm agent ideas for your department

- Identify needed tools
- Plan next steps

**5:00 - 5:15 PM: Final Q&A & Resources**
- Open forum
- Troubleshooting common issues

## Instructor Notes

### Preparation Checklist

**1 Week Before:**
- [ ] Test all exercises in clean environment
- [ ] Verify Google Cloud quotas and access
- [ ] Prepare backup examples
- [ ] Set up demo accounts
- [ ] Test Visual Builder on workshop network
- [ ] Prepare troubleshooting scenarios

**1 Day Before:**
- [ ] Final materials review
- [ ] Test all tools and examples
- [ ] Prepare contingency plans (internet issues, etc.)
- [ ] Print handouts (optional)
- [ ] Set up virtual meeting room
- [ ] Test screen sharing

**Day Of:**
- [ ] Arrive 30 minutes early
- [ ] Test AV equipment
- [ ] Start Visual Builder on demo machine
- [ ] Have backup examples ready
- [ ] Water/coffee ready
- [ ] Participant list

### Pacing Tips

**For Mixed Skill Levels:**
- Pair experienced with novices
- Provide bonus challenges for fast finishers
- Have extra examples ready
- Encourage peer helping

**If Running Behind:**
- Shorten discussion periods
- Instructor can demo instead of hands-on for Exercise 3
- Skip bonus challenges
- Extend breaks slightly to help strugglers catch up

**If Running Ahead:**
- Extend discussion periods
- Add bonus challenges
- Do additional examples
- More open Q&A time

### Common Issues & Solutions

**Issue:** Participants can't access Visual Builder
**Solution:** Have Python agent code ready as backup; troubleshoot during break

**Issue:** Google Cloud authentication problems
**Solution:** Pre-create demo accounts; use simulated tools that don't need cloud access

**Issue:** Wide range of skill levels
**Solution:** Buddy system; bonus challenges; instructor support

**Issue:** Specific use case questions beyond workshop scope
**Solution:** Capture for follow-up; schedule office hours; create parking lot
